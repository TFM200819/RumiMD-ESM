# IMPORTANT

## This is a soft fork from ShirokamiRyzen/RyzumiMD-ESM. All assets from Ryzumi Network has been replaced by my own assets. This fork has some additional plugins that I make to use personally and also some fixes to the existing plugins.
### The code remains licensed under GNU GPL v3

## NOTE BEFORE USING!!

- Edit all Owner Information on config.example.js
- Dont forget to rename `config.example.js` to `config.js`
- If the pairing code not appear in your terminal, simply go to `/sessions/creds.json` and you will see the code

## Requirements

- NodeJS V20+
- FFMPEG
- Imagemagick
- NodeJS basic knowledges (don't be an idiot)

#### Setup bot on your Machine

```bash
git clone https://github.com/TFM200819/Rumi-MD
cd Rumi-MD
npm i
```

---------

## Run

```bash
npm start
```

### Flags

- `--autocleartmp` : Automatically clear tmp folder every 5 minutes.
